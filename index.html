A-1. Domain of f(x) = (2x² - 5)/(x² + x - 6)

Step 1 — identify denominator. Denominator D(x) = x² + x - 6.

Step 2 — factor denominator.
x² + x - 6 = (x + 3)(x - 2).

Step 3 — domain restriction. Denominator cannot be zero, so exclude roots:
x ≠ 2, x ≠ -3.

Answer: Domain = ℝ ∖ {-3, 2}.

A-2. Sketch / analyze f(x) = {x², -2 ≤ x ≤ 0; 2-x, 0 < x ≤ 2}

Step 1 — left branch [-2,0]: function is x²
At x = -2: f(-2) = (-2)² = 4. (closed point)
At x = 0: f(0) = 0. (closed point, because 0 is included for left branch)

Step 2 — right branch (0,2]: function is 2-x
At x → 0⁺: 2-x → 2. But x = 0 is not included in the right branch → open spot at (0,2).
At x = 2: f(2) = 2-2 = 0. (closed point)

Step 3 — continuity at x = 0.
Left-hand value at 0: lim(x→0⁻) f(x) = 0.
Right-hand value at 0: lim(x→0⁺) f(x) = 2.
They are different ⇒ jump discontinuity at x = 0.

Step 4 — graphing notes:
Parabola segment from (-2,4) to (0,0), both filled.
An open dot at (0,2). A straight line from just right of 0 down to (2,0).

A-3. Compute lim(x→1) (x-1)/(x²-1)

Step 1 — factor denominator.
x² - 1 = (x - 1)(x + 1).

Step 2 — simplify (valid for x ≠ 1).
(x-1)/(x²-1) = (x-1)/[(x-1)(x+1)] = 1/(x+1).

Step 3 — take the limit.
lim(x→1) 1/(x+1) = 1/(1+1) = 1/2.

Answer: 1/2.

A-4. Differentiate y = x·tan(√x)

We compute y′ using product rule and chain rule.

Step 1 — product rule. Let u = x, v = tan(√x). Then
y′ = u′v + uv′.
So y′ = 1·tan(√x) + x·v′.

Step 2 — find v′ = d/dx tan(√x).
Let w = √x = x^(1/2). Then tan(w) derivative is sec²(w)·w′.
w′ = 1/(2√x).
So v′ = sec²(√x)·1/(2√x).

Step 3 — assemble.
y′ = tan(√x) + x·(sec²(√x)·1/(2√x)).

Step 4 — simplify the second term. x/(2√x) = √x/2.

Final answer:
y′ = tan(√x) + (√x/2)sec²(√x).

(This derivative is valid for x > 0 where √x is defined.)

A-5. Find critical points of y = 5x³ - 6x

Step 1 — derivative.
y′ = 15x² - 6.

Step 2 — set derivative to zero.
15x² - 6 = 0 ⇒ x² = 6/15 = 2/5.
x = ±√(2/5).

Step 3 — corresponding y-values (optional, but shown): Let s = √(2/5).
For x = s: x³ = s³ = s·s² = s·(2/5) = (2/5)s.
y(s) = 5·(2/5)s - 6s = 2s - 6s = -4s.
For x = -s: x³ = -(2/5)s,
y(-s) = 5·(-(2/5)s) - 6(-s) = -2s + 6s = 4s.

Answer: critical points (x-values) x = ±√(2/5).
If needed, points: (√(2/5), -4√(2/5)) and (-√(2/5), 4√(2/5)).

A-6. Lagrange's Mean Value Theorem (MVT)

Statement (clear form):
If f is continuous on the closed interval [a,b] and differentiable on the open interval (a,b), then there exists at least one c ∈ (a,b) such that

f′(c) = [f(b) - f(a)]/(b - a).

Interpretation: at some interior point the instantaneous rate of change equals the average rate of change over [a,b].

A-7. Differentiate y = x ln((x-1)/(x+1))

Step 0 — domain note. ln((x-1)/(x+1)) requires (x-1)/(x+1) > 0 and x ≠ -1. That condition holds for x > 1 and x < -1. Work on those intervals.

Step 1 — product rule. u = x, v = ln((x-1)/(x+1)).
y′ = u′v + uv′ = ln((x-1)/(x+1)) + x·v′.

Step 2 — compute v′.
v = ln(x-1) - ln(x+1) ⇒ v′ = 1/(x-1) - 1/(x+1).
Compute difference:
1/(x-1) - 1/(x+1) = [(x+1) - (x-1)]/[(x-1)(x+1)] = 2/(x²-1).

Step 3 — assemble.
y′ = ln((x-1)/(x+1)) + x·2/(x²-1).

Answer: y′ = ln((x-1)/(x+1)) + 2x/(x²-1), (x > 1 or x < -1).

A-8. Critical values of f(x) = 2x³ - 3x² - 36x

Step 1 — derivative.
f′(x) = 6x² - 6x - 36 = 6(x² - x - 6).
Factor quadratic:
x² - x - 6 = (x - 3)(x + 2).
So f′(x) = 6(x - 3)(x + 2).

Step 2 — critical points where f′(x) = 0.
x = 3, x = -2.

Step 3 — critical values f(x) at those x's.
f(3) = 2(27) - 3(9) - 36(3) = 54 - 27 - 108 = -81.
f(-2) = 2(-8) - 3(4) - 36(-2) = -16 - 12 + 72 = 44.

Answer: critical points x = -2, 3 with function values f(-2) = 44, f(3) = -81.

B-1. Find a,b so f continuous if f(x) = {(x³-8)/(x-2), x<2; ax²-bx+3, 2≤x≤3; 2x-a+b, x≥3}

Continuity requires matching values at boundary points x = 2 and x = 3.

Step 1 — simplify left expression for x ≠ 2. Factor numerator:
x³ - 8 = (x - 2)(x² + 2x + 4).
So for x ≠ 2,
(x³-8)/(x-2) = x² + 2x + 4.

Step 2 — left-hand limit at x = 2.
lim(x→2⁻) f(x) = 2² + 2·2 + 4 = 4 + 4 + 4 = 12.

Step 3 — continuity at x = 2 gives middle piece value at 2 equals 12.
a(2)² - b(2) + 3 = 12 ⇒ 4a - 2b + 3 = 12.
So 4a - 2b = 9 (Equation 1).

Step 4 — continuity at x = 3: middle piece at 3 equals right piece at 3.
a(3)² - b(3) + 3 = 2(3) - a + b.
Compute: 9a - 3b + 3 = 6 - a + b.
Bring all terms to left: 9a + a - 3b - b + 3 - 6 = 0 ⇒ 10a - 4b - 3 = 0,
i.e. 10a - 4b = 3 (Equation 2).

Step 5 — solve the linear system
{4a - 2b = 9
{10a - 4b = 3

Multiply Eq1 by 2: 8a - 4b = 18. Subtract Eq2:
(8a - 4b) - (10a - 4b) = 18 - 3 ⇒ -2a = 15 ⇒ a = -15/2.

Plug into Eq1:
4(-15/2) - 2b = 9 ⇒ -30 - 2b = 9 ⇒ -2b = 39 ⇒ b = -39/2.

Answer: a = -15/2, b = -39/2.

B-2. dy/dx for (i) x⁴+y⁴=16, (ii) x³+y³=3axy

(i) x⁴ + y⁴ = 16

Differentiate both sides w.r.t. x:
d/dx(x⁴) + d/dx(y⁴) = d/dx(16).
4x³ + 4y³(dy/dx) = 0.

Solve for dy/dx:
4y³(dy/dx) = -4x³ ⇒ dy/dx = -x³/y³.

(ii) x³ + y³ = 3axy

Differentiate both sides:
Left: 3x² + 3y²(dy/dx).
Right: 3a·d/dx(xy) = 3a(y + x(dy/dx)).

So: 3x² + 3y²(dy/dx) = 3a(y + x(dy/dx)).

Collect dy/dx terms to left:
3y²(dy/dx) - 3ax(dy/dx) = 3ay - 3x².

Factor dy/dx:
(dy/dx)(3y² - 3ax) = 3(ay - x²).

Divide:
dy/dx = (ay - x²)/(y² - ax).

B-3. Differentiate y = (a-x)²(b-x)³/(c-2x)³

Use logarithmic differentiation (cleanest).

Step 1 — take ln of both sides:
ln y = 2ln(a-x) + 3ln(b-x) - 3ln(c-2x).

Step 2 — differentiate with respect to x. Use d/dx ln(g(x)) = g′(x)/g(x):
d/dx ln(a-x) = -1/(a-x),
d/dx ln(b-x) = -1/(b-x),
d/dx ln(c-2x) = -2/(c-2x) (because derivative of c-2x is -2).

So: y′/y = -2/(a-x) - 3/(b-x) + 6/(c-2x).

Step 3 — multiply both sides by y (original function):
y′ = [(a-x)²(b-x)³/(c-2x)³][-2/(a-x) - 3/(b-x) + 6/(c-2x)].

This is a clean final form; you can expand to a single rational expression if desired.

B-4. Show continuity of f(x) = {1-x², x≤1; ln x, x≥1}

Step 1 — continuity on each piece.
1-x² is a polynomial — continuous everywhere.
ln x is continuous on (0,∞) — here used on [1,∞), so continuous there.

Step 2 — check at x = 1:
Left value f(1⁻) = 1-1² = 0.
Right value f(1⁺) = ln 1 = 0.
Both match, and both pieces include x = 1. Thus f is continuous at x = 1.

Conclusion: f is continuous for all real x (because left piece covers (-∞,1] and right piece covers [1,∞) and they agree at 1).

B-5. Let f(x) = {-√x, x<0; 3-x, 0≤x≤3; (x-3)², x>3}

Evaluate the limits:
lim(x→0⁻) f(x), lim(x→0⁺) f(x), lim(x→3⁻) f(x), lim(x→3⁺) f(x), lim(x→0) f(x), lim(x→3) f(x).

Compute one by one:

lim(x→0⁻) f(x)
For x→0⁻, use left formula f(x) = -√x. As x→0⁻, -√x→0⁺, so -√x→0.
Answer: 0.

lim(x→0⁺) f(x)
For x→0⁺, middle formula f(x) = 3-x. So lim(x→0⁺) 3-x = 3.
Answer: 3.

lim(x→0) f(x) — does it exist?
Left limit = 0, right limit = 3. They differ ⇒ the two-sided limit does not exist.

lim(x→3⁻) f(x)
For x→3⁻, middle formula 3-x gives 3-3 = 0. So left-hand limit at 3 is 0.

lim(x→3⁺) f(x)
For x→3⁺, right formula (x-3)²→0. So right-hand limit equals 0.

lim(x→3) f(x) — two-sided limit exists and equals 0. Also the middle piece value f(3) = 3-3 = 0 so the function is continuous at x = 3.

Summary:
lim(x→0⁻) f(x) = 0.
lim(x→0⁺) f(x) = 3.
lim(x→0) f(x) does not exist.
lim(x→3⁻) f(x) = 0.
lim(x→3⁺) f(x) = 0.
lim(x→3) f(x) = 0 (function continuous at 3).

B-6. Tangent and normal to f(x) = (x²-1)/(x²+x+1) at (1,0)

Step 1 — verify point on curve
Plug x = 1: f(1) = (1-1)/(1+1+1) = 0/3 = 0. OK.

Step 2 — derivative (quotient rule).
If u(x) = x²-1, v(x) = x²+x+1, then
f′(x) = (u′v - uv′)/v².

Compute: u′ = 2x, v′ = 2x+1.

So f′(x) = [2x(x²+x+1) - (x²-1)(2x+1)]/(x²+x+1)².

Step 3 — evaluate at x = 1.
Numerator: 2(1)(1+1+1) - (1-1)(2+1) = 2·3 - 0 = 6.
Denominator: (1+1+1)² = 3² = 9.
So f′(1) = 6/9 = 2/3.

Step 4 — tangent line equation. Point-slope form at (1,0) with slope 2/3:
y - 0 = (2/3)(x - 1) ⇒ y = (2/3)x - 2/3.

Step 5 — normal line equation. Normal slope m_n = -1/m_t = -3/2. So:
y - 0 = -(3/2)(x - 1) ⇒ y = -(3/2)x + 3/2.

B-7. Absolute max & min of f(x) = 3x⁴-16x³+18x² on [-1,4]

Step 1 — derivative.
f′(x) = 12x³-48x²+36x = 12x(x²-4x+3) = 12x(x-1)(x-3).

Step 2 — critical points in interval. Solve f′(x) = 0:
x = 0, 1, 3.

Step 3 — evaluate f at endpoints and critical points.
f(-1) = 3(-1)⁴-16(-1)³+18(-1)² = 3+16+18 = 37.
f(0) = 0.
f(1) = 3-16+18 = 5.
f(3) = 3(81)-16(27)+18(9) = 243-432+162 = -27.
f(4) = 3(256)-16(64)+18(16) = 768-1024+288 = 32.

Step 4 — compare values.
Largest value is 37 at x = -1.
Smallest value is -27 at x = 3.

Answer: Absolute max 37 at x = -1, Absolute min -27 at x = 3.

B-8. Increasing / decreasing for f(x) = 3x⁴-4x³-12x²+5

Step 1 — derivative.
f′(x) = 12x³-12x²-24x = 12x(x²-x-2) = 12x(x-2)(x+1).

Step 2 — critical points at x = -1, 0, 2.

Step 3 — sign analysis (test intervals):
For x < -1: take x = -2: factors (x) negative, (x-2) negative, (x+1) negative ⇒ product negative → f′ < 0 (decreasing).
For -1 < x < 0: take -0.5: signs (x) negative, (x-2) negative, (x+1) positive ⇒ product positive → f′ > 0 (increasing).
For 0 < x < 2: take 1: signs: +, -, + → negative → f′ < 0 (decreasing).
For x > 2: take 3: all factors positive → f′ > 0 (increasing).

Answer:
f is increasing on (-1,0) ∪ (2,∞).
f is decreasing on (-∞,-1) ∪ (0,2).

B-9. If y^x = x^y, prove dy/dx = y(y-x ln y)/[x(x-y ln x)]

(Assume x > 0, y > 0 so logs are defined.)

Step 1 — take natural log both sides:
x ln y = y ln x.

Step 2 — differentiate both sides w.r.t. x. Use product rule on both sides:
Left derivative: 1·ln y + x·(1/y)(dy/dx).
Right derivative: (dy/dx)·ln x + y·(1/x).

So: ln y + (x/y)(dy/dx) = ln x(dy/dx) + y/x.

Step 3 — gather dy/dx terms on left:
(x/y)(dy/dx) - ln x(dy/dx) = y/x - ln y.

Factor dy/dx:
(dy/dx)(x/y - ln x) = y/x - ln y.

Step 4 — write RHS and LHS with common factors to reach target form. Multiply numerator and denominator appropriately:
dy/dx = (y/x - ln y)/(x/y - ln x).

Multiply numerator and denominator by xy:
dy/dx = (y² - xy ln y)/(x² - xy ln x) = y(y - x ln y)/[x(x - y ln x)].

Result: dy/dx = y(y - x ln y)/[x(x - y ln x)].

B-10. For f(x) = 2x³+3x²-36x: (i) critical points, (ii) intervals of inc/dec, (iii) local max/min, (iv) concavity & inflection

Step 1 — derivative.
f′(x) = 6x²+6x-36 = 6(x²+x-6) = 6(x+3)(x-2).

(i) Critical points: Solve f′ = 0: x = -3, x = 2.

(ii) Intervals of increase / decrease (sign test):
For x < -3 (take -4): factors (x+3) < 0, (x-2) < 0 ⇒ product + ⇒ f′ > 0 (increasing).
For -3 < x < 2 (take 0): (x+3) > 0, (x-2) < 0 ⇒ product - ⇒ f′ < 0 (decreasing).
For x > 2: product + ⇒ f′ > 0 (increasing).
So f increases on (-∞,-3) ∪ (2,∞), decreases on (-3,2).

(iii) Local extrema:
At x = -3: f′ changes +→- ⇒ local maximum.
At x = 2: f′ changes -→+ ⇒ local minimum.
Compute values:
f(-3) = 2(-27)+3(9)-36(-3) = -54+27+108 = 81.
f(2) = 16+12-72 = -44.
So local max 81 at x = -3; local min -44 at x = 2.

(iv) Concavity & inflection.
Second derivative: f′′(x) = 12x+6.
Set f′′(x) = 0 → 12x+6 = 0 ⇒ x = -1/2.
For x < -1/2, f′′ < 0 ⇒ concave down.
For x > -1/2, f′′ > 0 ⇒ concave up.
Inflection point: x = -1/2, value:
f(-1/2) = 2(-1/8)+3(1/4)-36(-1/2) = -1/4+3/4+18 = 1/2+18 = 37/2.
So inflection point (-1/2, 37/2).

B-11. For y = x⁴-4x³: (i) critical points, (ii) inc/dec, (iii) local extrema, (iv) concavity & inflection

Step 1 — derivative.
y′ = 4x³-12x² = 4x²(x-3).

(i) Critical points: y′ = 0 ⇒ x = 0 (double root) and x = 3.

(ii) Intervals (sign):
For x < 0: x² > 0, x-3 < 0 ⇒ y′ < 0 (decreasing).
For 0 < x < 3: x² > 0, x-3 < 0 ⇒ y′ < 0 (decreasing) — no sign change at 0 because multiplicity 2.
For x > 3: y′ > 0 (increasing).
So decreasing on (-∞,3), increasing on (3,∞). (At x = 0 derivative zero but no extrema.)

(iii) Local extrema:
x = 0: derivative zero but no sign change ⇒ neither max nor min.
x = 3: derivative changes negative→positive ⇒ local minimum at x = 3. Value y(3) = 3⁴-4·3³ = 81-108 = -27.

(iv) Concavity & inflection.
Second derivative: y′′ = 12x²-24x = 12x(x-2).
Zeros at x = 0 and x = 2.
x < 0: y′′ > 0 ⇒ concave up.
0 < x < 2: y′′ < 0 ⇒ concave down.
x > 2: y′′ > 0 ⇒ concave up.
Inflection points at x = 0 and x = 2:
y(0) = 0. So inflection (0,0).
y(2) = 2⁴-4·2³ = 16-32 = -16. So inflection (2,-16).

B-12. If f(x) = xe^x, find f′(x) and f^(n)(x) (the nth derivative)

Step 1 — first derivative (product rule).
f′(x) = e^x + xe^x = e^x(1+x).

Step 2 — compute a couple to see pattern:
f′′(x) = d/dx[e^x(1+x)] = e^x(1+x) + e^x = e^x(x+2).
f^(3)(x) = d/dx[e^x(x+2)] = e^x(x+2) + e^x = e^x(x+3).

Step 3 — observe pattern and prove by induction.
Conjecture: f^(n)(x) = e^x(x+n).
Base (n=1): f′(x) = e^x(x+1) — true.
Inductive step: assume f^(n)(x) = e^x(x+n). Then
f^(n+1)(x) = d/dx[e^x(x+n)] = e^x(x+n) + e^x = e^x(x+n+1),
so the formula holds.

Answer: f′(x) = e^x(x+1), f^(n)(x) = e^x(x+n).
