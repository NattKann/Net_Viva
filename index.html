1. What is Data Communication?

Data communication refers to the exchange of data between two devices through some form of transmission medium such as a wire cable or wireless channel.

2. What are the key components of a data communication system?

Message, Sender, Receiver, Transmission medium, Protocol

3. What are protocols in networking?

Protocols are a set of rules that govern data communication — specifying how data is transmitted, formatted, and received.

4. What is the difference between a Hub, Switch, and Router?

Hub: Broadcasts data to all ports (Layer 1)
Switch: Forwards data to specific MAC addresses (Layer 2)
Router: Forwards packets based on IP addresses (Layer 3)

5. What are the layers of the OSI model?

Physical, Data Link, Network, Transport, Session, Presentation, Application

6. Which layer handles routing?

The Network layer (Layer 3) handles routing and logical addressing (IP).

7. What is the main function of the Transport layer?

To provide reliable process-to-process data delivery (e.g., TCP, UDP).

8. What are the main types of transmission media?

Guided: Twisted pair, coaxial cable, fiber optic
Unguided: Radio, microwave, infrared

9. What is the difference between TCP and UDP?

TCP: Connection-oriented, Reliable, Slower (HTTP, FTP)
UDP: Connectionless, Unreliable, Faster (DNS, VoIP)

10. What is IP addressing?

IP addressing provides a unique logical address to each device on a network, enabling proper routing of packets.

11. What is the difference between IPv4 and IPv6?

IPv4: 32 bits, Decimal format (192.168.1.1), ~4 billion addresses
IPv6: 128 bits, Hexadecimal format (2001:db8::1), Almost unlimited addresses

12. What is a subnet mask?

A subnet mask divides the IP address into network and host portions, determining the size of the network.

13. What is a router?

A router is a device that connects multiple networks and forwards data packets based on IP addresses.

14. What is the purpose of the routing table?

It stores paths and rules for forwarding packets to the correct destination network.

15. What are static and dynamic routing?

Static Routing: Routes manually configured by the administrator.
Dynamic Routing: Routes learned automatically using routing protocols (e.g., RIP, OSPF, EIGRP).

16. What is RIP (Routing Information Protocol)?

A distance-vector routing protocol that uses hop count as the metric and sends updates every 30 seconds.

17. What is the maximum hop count allowed in RIP?

15 hops. A hop count of 16 means the destination is unreachable.

18. What is OSPF?

Open Shortest Path First — a link-state routing protocol that uses Dijkstra's algorithm to calculate the shortest path.

19. What are the differences between RIP and OSPF?

RIP: Distance Vector, Hop Count metric, Slow convergence, 15 hop limit
OSPF: Link State, Cost (Bandwidth) metric, Fast convergence, No hop limit

20. What is a default route?

A route used by the router when no other route matches the destination IP address.

21. What is a gateway of last resort?

The router used as a last option to forward packets when no specific route is known — often configured as a default route.

22. What is meant by "ping" in networking?

ping checks network connectivity by sending ICMP Echo Request packets and waiting for Echo Reply.

23. What is ARP?

Address Resolution Protocol maps an IP address to its physical MAC address.

24. What is DNS?

Domain Name System translates human-readable domain names (e.g., google.com) into IP addresses.

25. What is DHCP?

Dynamic Host Configuration Protocol automatically assigns IP addresses to devices on a network.

26. What is NAT?

Network Address Translation converts private IP addresses to a public IP for internet communication.

27. What is QoS in networking?

Quality of Service ensures priority for important traffic (like voice or video) to reduce latency and packet loss.

28. What is the difference between unicast, multicast, and broadcast?

Unicast: One-to-one communication
Multicast: One-to-many selected nodes
Broadcast: One-to-all in the network

29. What is congestion in a network?

When the load on the network exceeds its capacity, leading to delay and packet loss.

30. What is meant by network topology?

The physical or logical arrangement of network devices — e.g., Bus, Star, Ring, Mesh, Hybrid.
